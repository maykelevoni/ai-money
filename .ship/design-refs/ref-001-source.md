# Healthline Review Page Pattern

## Layout Structure
1. Header: title + author byline + review credentials
2. Fast Facts summary box (quick verdict)
3. Product grid with category labels
4. Detailed review sections per product
5. Comparison table
6. FAQ section
7. Related content

## Review Component Per Product
- Star rating badge (top-right)
- Price callout box
- Three-column spec layout
- Tester narrative (1-2 paragraphs)
- Pros/cons formatted list
- Product details table

## CTA Pattern
- Primary CTA immediately after specs
- Discount/trust callout above price
- Multiple inline CTAs throughout

## Trust Signals
- Medical review attribution
- Third-party testing badges
- Certification badges
- Author credentials

## Colors
- White background, gray accent boxes
- Green for positive/affirmative elements
- Blue hyperlinks
